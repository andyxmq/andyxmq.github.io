# 创建对象

## 对象简介

```
    定义：对象是js的基本数据类型，对象是一种复合值：它将很多值（原始值或者其他对象）聚合在一起；可以通过名字
    访问这些值，对象也可以看做是属性的无序集合，每个属性都是一个名/值对，属性名是字符串,因此我们可以把对象看
    做是字符串到值的映射；其他叫法：散列、散列表、字典、关联数组；

    js对象除了有自身的属性，还可以从原型的对象继承属性;
    注意：除字符串、数字、true、false、null、undefined之外,js中的值都是对象；对象是可变的，所以是通过对象
    的引用而非值来操作对象；

    常见的用法: 创建(create)、设置(set)、查找(query)、删除(delete)、检测(test)和枚举(enumerate）的属性；
    属性包含名字和值，属性名可以包含空字符串在内的任意字符串；对象中不能存在两个同名的属性，值可以时任意的js值，也可以是getter或setter函数;

    属性的特性：可写、可枚举、可配置
    二类属性：自有属性、继承属性；

    对象特性：对象的原型指向另外一个对象，本对象的属性继承自他的原型对象；
        对象的类(class)是一个标识对象类型的字符串；
        对象的扩展标记指明了是否可以向该对象添加新属性；
    三类对象：内置对象（数组、函数、日期等）、宿主对象、自定义对象
```
