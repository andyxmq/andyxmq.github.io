# 删除属性
> delete操作符
```
    delete运算符可以删除对象的属性，操作符应当是一个属性访问表达式；
    注意：delete只是断开了属性和宿主对象的联系；而不会去操作属性的属性；
    var a = {p:{x:1},x:2};
    b = a.p;
    delete a.p;
    console.log(a,b.x);
    由于删除的属性的引用还在，极易造成内存泄漏；

    注意：delete只能删除自有属性，不能删除继承属性；当delete表达式删除
    成功或没有任何副作用（比如删除不存在得属性）时，他返回true,delete 
    后不是一个访问表达式，delete同样返回true;
    
    delete 不能删除那些可配置性为false的属性；某些内置对象的属性是不可配置的，
    比如通过变量声明或函数声明创建的全局对象的属性。

    非严格模式下：删除全局对象可配置属性，可以省略全局对象的引用
    严格模式下：则会报错
```
